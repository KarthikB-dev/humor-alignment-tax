Place raw joke datasets here:

- r_jokes.json       — Reddit r/Jokes dataset (JSON lines: {"title": ..., "body": ...})
- one_liners.txt     — 16k one-liner dataset (one joke per line)
- oogiri.json         — Oogiri dataset (JSON array with prompt/response fields)
- ruozhiba.json       — Ruozhiba dataset (JSON array with question/answer fields)

Download links:
- r/Jokes: https://github.com/taivop/joke-dataset
- One-liners: https://github.com/taivop/joke-dataset
- Oogiri: https://github.com/JokerHYC/Oogiri-dataset
- Ruozhiba: https://huggingface.co/datasets/mrzjy/ruozhiba
